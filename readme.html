Watch the Demo Video to see the plugin in action.

Installation Guide

  1. Navigate to the Plugin Folder
    • Go to ./wp-content/plugins/custom-tabs-plugin.
  2. Download the Folder
    • Download the entire custom-tabs-plugin folder.
  3. Insert the Folder in Your WordPress Directory
    • Place the downloaded custom-tabs-plugin folder into your WordPress directory at ./wp-content/plugins/.
  4. Activate the Plugin
    • You can now see the plugin named custom-tabs-plugin in your WordPress plugins menu.

How to Use the Plugin

  1. Activate the Plugin
    • Go to the WordPress plugins menu and activate the custom-tabs-plugin.
  2. Navigate to the Settings Tab
    • Go to Settings > Custom Tabs.
  3. Manage the Content
    • In the Custom Tabs section, you can manage the content as desired.
  4. Save Changes
    • After inserting the content, click the Save Changes button.

How to Use the Plugin Shortcode

  1. Go to Any Page
    • Navigate to any page of your choice in WordPress.
  2. Insert a Shortcode Element
    • Add a shortcode element to the page.
  3. Use the Shortcode
    • The shortcode for this plugin is [custom_tabs_value].
  4. View the Page
    • View the page to see the custom tab component in action.
